RoundRobinPartitionRouter

The round-robin partition messages router, which is the default router. If a key is provided, the producer will hash the key and publish the message to a particular partition. If a key is not provided, the producer will publish messages across all partitions in a round-robin fashion to achieve maximum throughput.

SinglePartitionRouter

The single partition messages router. If a key is provided, the producer will hash the key and publish the message to a particular partition. If a key is not provided, the producer will randomly pick one single partition and publish all messages to that partition.
